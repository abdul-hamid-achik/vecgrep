# Development Guide

## Quick Start

```bash
# Check your environment
task doctor

# Setup everything
task setup

# Start developing
task dev
```

## Daily Workflow

```bash
task dev          # Hot reload development
task check        # Run before committing
task ship         # Full CI pipeline locally
```

## Debugging

```bash
task wtf          # What's broken?
task doctor       # Environment check
```

## Release

```bash
task release -- v1.0.0
```
